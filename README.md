# Echelix Playwright Automation

Playwright end-to-end test automation framework — setup in progress.

## Project Structure

```
echelix-playwright-automation/
├── tests/                  # Test specs
├── api/                    # API clients
├── pages/                  # Page Object Models
├── services/               # External service integrations (Gmail API)
├── utils/                  # Utility helpers
├── playwright.config.ts    # Playwright configuration
├── tsconfig.json           # TypeScript configuration
├── .env.example            # Environment variable template
└── README.md               # This file
```

## Tech Stack

- Playwright (TypeScript)
- Node.js
- Gmail API
- Playwright HTML Reporter
